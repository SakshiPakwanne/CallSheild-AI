# 🛡️ AI CallShield

AI CallShield is a real-time AI system that detects and prevents fraud calls during the conversation.

## 🚀 Features
- Real-time speech recognition
- Fraud detection using ML + rules
- Truecaller-like UI
- AI voice response
- Automatic call blocking

## 🧠 Tech Stack
- Python
- Streamlit
- SpeechRecognition
- Machine Learning (TF-IDF + Model)

## 🎯 Idea
Unlike Truecaller, which identifies callers,  
AI CallShield understands the conversation and protects users in real-time.
